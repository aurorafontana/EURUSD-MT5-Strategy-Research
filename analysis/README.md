# Analysis

This directory contains research notes and, in future revisions, reproducible analysis scripts or notebooks.

Priority analyses:

- MAE/MFE by Magic Number;
- FIRST vs ADD performance;
- basket reconstruction;
- stop-loss sensitivity;
- take-profit sensitivity;
- hour-of-day/day-of-week analysis;
- Friday/weekend exposure;
- market-regime classification;
- walk-forward validation.

All analyses should document the input dataset, timezone assumptions, code version, and whether the method is static or dynamically reconstructs strategy state.
